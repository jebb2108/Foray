# Foray

```bash
npm install
npm run dev
```

## Project Structure

```plaintext
src/
├── app/          # routing and application composition
├── constants/    # shared constants
├── pages/        # route-level screens
├── styles/       # global Ionic and Foray styles
├── theme/        # design tokens
├── types/        # shared TypeScript models
├── utils/        # reusable helpers
├── App.tsx
└── main.tsx
```
