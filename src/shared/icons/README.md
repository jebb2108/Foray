# Локальные иконки

Директория содержит SVG иконки, которые используются интерфейсом Foray.

Иконки скопированы из Ionicons 7.4.0 и подключаются через `index.ts`.
Прямые импорты из `ionicons/icons` в коде приложения не используются.
